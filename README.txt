To run and compile the program you need Python 2.7 and pip installed. 
First create a folder named data and copy the files hsa_hg19_Rybak2015.bed, hg19.fa and all_exons.bed into this folder.
Next run  in the folder training_data the file training_data.py. This will generate the positive and negative
examples for use in the AI algorithm
Next run predict.py without arguments.
